# 🐰 Ферма невидимых кроликов

Интерактивный интерфейс для фермера, обнаруживающего невидимых кроликов по косвенным сигналам.

## 📋 О проекте

Это тестовое задание для позиции AI-first Developer. Проект представляет собой интерактивное веб-приложение, которое помогает фермеру:

- Отслеживать косвенные сигналы присутствия кроликов
- Оценивать количество невидимых кроликов
- Получать рекомендации по действиям
- Анализировать активность по локациям и времени

## 🚀 Быстрый старт

### Предварительные требования

- Node.js (версия 18 или выше)
- npm (версия 8 или выше)

### Установка

1. Клонируйте репозиторий:

   ```bash
   git clone https://github.com/inspectorrrrr/fantastic-invention.git
   cd fantastic-invention
   ```

2. Установите зависимости:

   ```bash
   npm install
   ```

3. Запустите сервер разработки:

   ```bash
   npm run dev
   ```

4. Откройте в браузере: [http://localhost:5173](http://localhost:5173)

### Сборка для продакшена

```bash
npm run build
```

Собранные файлы будут в папке `dist/`.

## 🧮 Алгоритм расчёта

### Количество кроликов

```text
rabbits = Σ (count × intensity × weight) / 10
```

Где:
- `count` - сколько раз произошло событие
- `intensity` - интенсивность события (1-10)
- `weight` - вес типа события (настраивается)

### Уверенность системы

Рассчитывается на основе 4 факторов:

1. Разнообразие типов событий (до 40%)
2. Количество локаций (до 30%)
3. Общее количество событий (до 20%)
4. Средняя интенсивность (до 10%)
